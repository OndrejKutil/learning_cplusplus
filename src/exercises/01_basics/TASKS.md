# Level 1 - Basics

Goal: get comfortable with syntax, input/output, variables, and simple arithmetic.

## Exercise 1: Hello + Name
File: `ex01_hello_name.cpp`
- Ask the user for their name.
- Print: `Hello, <name>!`
- Add one extra line showing the name length.

## Exercise 2: Temperature Converter
File: `ex02_temperature_converter.cpp`
- Read a temperature in Celsius.
- Convert to Fahrenheit using: `F = C * 9 / 5 + 32`.
- Print both values with labels.
- Bonus: also convert Fahrenheit back to Celsius.

## Exercise 3: Basic Calculator
File: `ex03_basic_calculator.cpp`
- Read two numbers and one operator (`+`, `-`, `*`, `/`).
- Print the result.
- Handle division by zero safely.

## Completion Check
- You can explain each line in your solution.
- Your program behaves correctly with normal and edge inputs.
- You used functions at least once (even small helper functions).
