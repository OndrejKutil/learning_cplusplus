# Level 2 - Control Flow

Goal: practice if/else logic, loops, and basic validation.

## Exercise 1: Number Guessing Game
File: `ex01_number_guessing.cpp`
- Hardcode a secret number.
- Repeatedly ask the user to guess until correct.
- Print hints: `too low` or `too high`.
- Show number of attempts at the end.

## Exercise 2: Prime Checker
File: `ex02_prime_checker.cpp`
- Read one integer `n`.
- Print whether it is prime.
- Handle values less than 2.
- Bonus: print the first 20 prime numbers.

## Exercise 3: FizzBuzz Plus
File: `ex03_fizzbuzz_plus.cpp`
- Print numbers from 1 to `N`.
- Replace multiples of 3 with `Fizz`, multiples of 5 with `Buzz`, and both with `FizzBuzz`.
- Bonus: add custom divisors and words.

## Completion Check
- You can choose between `for`, `while`, and `do-while` appropriately.
- You can justify your loop conditions.
- Inputs like negatives and zero are handled cleanly.
