# Level 3 - Collections and Algorithms

Goal: build confidence with vectors, strings, and algorithmic thinking.

## Exercise 1: Min, Max, Average
File: `ex01_min_max_avg.cpp`
- Read `N`, then read `N` numbers into a vector.
- Compute minimum, maximum, and average.
- Print all three values clearly.

## Exercise 2: Reverse String
File: `ex02_reverse_string.cpp`
- Read a full line of text.
- Reverse it manually (without `std::reverse` first).
- Bonus: check if the input is a palindrome (ignore spaces and case).

## Exercise 3: Sort and Search
File: `ex03_sort_and_search.cpp`
- Read numbers into a vector.
- Implement bubble sort manually.
- Ask for a target value and perform linear search.
- Bonus: after sorting, implement binary search manually.

## Completion Check
- You understand indexing and bounds.
- You can describe time complexity informally.
- You can compare manual implementations with STL alternatives.
